ACL for Proxy Engine

## Install

```sh
$ npm install --save trailpack-proxy-permissions
```

## Configure

```js
// config/main.js
module.exports = {
  packs: [
    // ... other trailpacks
    require('trailpack-proxy-permissions')
  ]
}
```
